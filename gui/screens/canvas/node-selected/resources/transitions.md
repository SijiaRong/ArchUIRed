# Canvas Node-Selected — Transitions

| Trigger | Target State |
|---------|-------------|
| Click empty canvas space | `idle` |
| Single-click a different node | `node-selected` (new selection, replaces current) |
| Double-click the selected node | `drilled` |
| Click `[↗]` on the selected node header | `drilled` |
| Press `Escape` | `idle` |
